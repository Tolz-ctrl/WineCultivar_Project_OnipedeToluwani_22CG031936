ğŸ¯ PROJECT OVERVIEW:
================================================================================

WHAT IT DOES:
  Predicts wine cultivar (origin) based on 6 chemical properties

DATASET:
  UCI Wine Dataset from sklearn (178 samples, 3 cultivars)

FEATURES USED:
  1. Alcohol
  2. Malic Acid
  3. Ash
  4. Alcalinity of Ash
  5. Magnesium
  6. Flavanoids

ALGORITHM:
  Logistic Regression (multiclass classification)

ACCURACY:
  ~95-100% on test set

INTERFACE:
  Flask web application with user-friendly form
