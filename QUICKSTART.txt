ğŸ”§ TROUBLESHOOTING:
-------------------

Problem: "ModuleNotFoundError: No module named 'pandas'"
Solution: Run: pip install -r requirements.txt

Problem: "Model or scaler not loaded"
Solution: Train the model first: python model/train_model.py

Problem: "Port 5000 already in use"
Solution: Edit app.py, change port 5000 to 5001 in last line

Problem: Jupyter notebook won't open
Solution: Install Jupyter: pip install jupyter notebook
