DEPLOYMENT INSTRUCTIONS:

Option 1: Deploy to Render
1. Create a free account at https://render.com
2. Create a new Web Service
3. Connect your GitHub repository
4. Set build command: pip install -r requirements.txt
5. Set start command: gunicorn app:app
6. Add gunicorn to requirements.txt: gunicorn==21.2.0
7. Deploy and copy the URL here

Option 2: Deploy to Railway
1. Create a free account at https://railway.app
2. Create a new project from GitHub repo
3. Railway will auto-detect Flask app
4. Deploy and copy the URL here

Option 3: Local Deployment
1. Install dependencies: pip install -r requirements.txt
2. Run the notebook to train model: jupyter notebook model/model_building.ipynb
3. Start Flask app: python app.py
4. Access at: http://localhost:5000
